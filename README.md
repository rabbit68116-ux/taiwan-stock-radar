Taiwan stock quantitative research framework architecture draft and implementation workspace.

The project is intended to evolve into an open-source framework for:

- Taiwan stock market data pipelines
- Feature engineering and radar scoring
- Signal generation and backtesting
- Dashboard and daily scan outputs
