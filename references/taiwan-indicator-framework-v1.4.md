# Taiwan Indicator Framework v1.4

Version: `v1.4`

這份文件整理 `Taiwan Stock Radar v1.4` 的核心台股指標框架。  
它的目的不是把所有可能的指標都列滿，而是整理出最值得先建立、最能支撐 AI 判斷的台股訊號系統。

## 1. 設計原則

`v1.4` 的觀點很直接：

- 不要只靠單一技術指標
- 不要只看主力進出軟體
- 不要只看新聞標題
- 台股應該同時看技術面、量價面、籌碼面、基本面、事件面

對單股分析而言，最穩定的結構是：

1. 趨勢
2. 動能
3. 量價
4. 籌碼
5. 基本面
6. 事件

## 2. 最值得先建立的核心技術指標

### A. 均線系統

最實用的是：

- `5MA / 10MA`：短線節奏
- `20MA`：波段強弱分界
- `60MA`：中期趨勢
- `120MA / 240MA`：長線方向

實戰解讀：

- 股價站上 `20MA`、`60MA`，且均線走揚：偏多
- `5MA` 上穿 `20MA`：短線轉強
- 跌破 `20MA` 且量增：波段轉弱要小心
- `60MA` 之上找買點，`60MA` 之下少追價

### B. 成交量與量價關係

台股很吃量，很多沒有量的突破都是假的。

重點欄位：

- 突破前高是否放量
- 回檔是否縮量
- 下跌是否爆大量
- `5日`、`20日` 均量是否同步上升

實戰解讀：

- 價漲量增：健康上攻
- 價漲量縮：可能只是反彈
- 價跌量縮：正常整理
- 價跌量增：賣壓明顯

### C. MACD

MACD 在台股中期波段很好用。

實戰解讀：

- `DIF` 上穿 `MACD`：偏多
- `0` 軸之上黃金交叉強於 `0` 軸下黃金交叉
- 柱狀體翻紅擴大：動能增強
- 高檔背離：常是轉弱警訊

### D. RSI

RSI 適合看短中線過熱 / 過弱。

常用區間：

- `RSI > 70`：偏熱
- `RSI < 30`：偏弱
- `50` 上方偏強，`50` 下方偏弱

注意：

- 強勢股可以長時間維持高 RSI
- 不應單看 RSI 就放空或賣掉強勢股

### E. KD

KD 在台股短線交易很常用，但容易鈍化。

適合：

- 短線
- 區間盤
- 抓回檔反彈

不適合：

- 單獨拿來判斷主升段結束

### F. 布林通道

適合看波動收斂與擴張。

實戰解讀：

- 收斂後開口放大：可能要出方向
- 沿上軌走：強勢趨勢
- 跌破中軌後站不回：轉弱機率提高

## 3. 台股特別重要的籌碼資訊

### A. 三大法人

重點看：

- 外資連買 / 連賣
- 投信連買
- 自營商方向
- 法人買超是否伴隨股價創高

實戰解讀：

- 投信連買對中小型股常很有用
- 外資方向對權值股、ETF、指數影響更大

### B. 融資融券

重點看：

- 融資增加但股價不漲：籌碼可能變亂
- 融資大減、股價止穩：有時是洗盤後轉強
- 融券高、股價又強：容易嘎空

### C. 借券賣出 / 借券餘額

這在台股近年非常重要。

實戰解讀：

- 借券賣出增加：有人在偏空布局
- 股價強、借券高：也可能形成回補行情
- 看空時，融券與借券應一起看

## 4. 基本面資訊

### A. 本益比、殖利率、股價淨值比

用途：

- 高估 / 低估初步篩選
- 高股息股篩選
- 景氣循環股估值比較
- 同產業相對便宜 / 昂貴比較

注意：

- 成長股不能只看低本益比
- 景氣循環股低本益比有時反而是高點附近

### B. 月營收

特別看：

- `MoM`
- `YoY`
- 連續 `2-3` 個月轉強
- 營收創高但股價未動

### C. 財報

至少看：

- `EPS`
- 毛利率
- 營益率
- `ROE`
- 現金流
- 存貨變化
- 應收帳款變化

### D. 股利與殖利率

台股很多中長線與除息交易會重視：

- 現金股利穩定性
- 配發率
- 除權息前後走勢

## 5. 事件與消息面

至少追蹤：

- 法說會
- 重大訊息公告
- 董事會決議
- 財報公布日
- 月營收公布日
- 除權息日期
- 產業新聞
- 美股 / 費半 / ADR 影響

對台股來說，公開資訊觀測站是核心資料來源。

## 6. 最推薦的觀察欄位結構

### A. 趨勢面

- 收盤價
- `5/10/20/60/120` 日均線
- 是否站上 `20MA / 60MA`
- `20MA` 斜率
- `60MA` 斜率
- `52` 週新高 / 新低距離

### B. 動能面

- `RSI(14)`
- `KD(9,3,3)`
- `MACD`
- 布林通道位置
- 近 `5` 日報酬
- 近 `20` 日報酬

### C. 量價面

- 當日成交量
- `5日均量 / 20日均量`
- 量比
- 價漲量增 / 價跌量增判定
- 是否放量突破平台

### D. 籌碼面

- 外資買賣超
- 投信買賣超
- 自營商買賣超
- 融資增減
- 融券增減
- 借券賣出增減
- 連買 / 連賣天數

### E. 基本面

- 月營收 `YoY`
- 月營收 `MoM`
- `EPS`
- 本益比
- 殖利率
- 股價淨值比
- `ROE`
- 毛利率 / 營益率

### F. 事件面

- 法說會日期
- 財報公布日期
- 月營收公布日期
- 除權息日期
- 重大訊息有無
- 所屬產業題材

## 7. 不同交易風格該看什麼

### 短線

優先看：

- `5MA / 10MA / 20MA`
- 成交量
- `KD`
- `RSI`
- 法人短期買賣超

### 波段

優先看：

- `20MA / 60MA / 120MA`
- `MACD`
- 量價突破
- 投信 / 外資趨勢
- 月營收與財報

### 存股 / 中長線

優先看：

- 殖利率
- 配息穩定性
- 本益比
- `ROE`
- 現金流
- 產業趨勢
- 大盤位置

## 8. 最實用的三種組合

### 趨勢突破

- `20MA` 上彎
- 股價站上 `60MA`
- 放量突破前高
- `MACD` 翻正
- 外資或投信同步買超

### 回檔承接

- 多頭排列
- 回測 `20MA` 或 `60MA`
- 回檔量縮
- `KD` 低檔黃金交叉
- 法人沒有明顯轉賣

### 價值 + 技術

- 本益比合理
- 殖利率不差
- 月營收 `YoY` 轉強
- 股價重新站回 `60MA`
- 量能開始放大

## 9. 最後原則

下面這些不建議單獨使用：

- 只看 `KD`
- 只看 `RSI`
- 只看本益比
- 只看主力進出軟體
- 只看新聞標題

`v1.4` 的真正目的不是讓 agent 記住更多指標名稱，而是讓 agent 能用台股最實用的欄位，做出更像資深分析師的判讀。
