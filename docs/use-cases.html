Use cases

How Taiwan Stock Radar is applied in practice

Taiwan Stock Radar is designed for focused single-stock work. Depending on the task, the same council framework can support a fast judgment, a deeper committee memo, or a review of a previous thesis.

Single-stock judgment

Used when the user wants a fast but disciplined answer on one ticker. The council returns direction bias, key evidence, risk flags, and actionable zones.

Deep-dive committee memo

Used when the user wants a higher-quality research note. The skill expands into specialist briefs, disagreement review, scenario analysis, and a fuller action plan.

Pre-trade review

Used before entry. The council tests whether structure, flow, event timing, and risk all support the setup strongly enough to take action.

Post-call review

Used after a thesis has played out. The skill compares the original logic with later market behavior and identifies which specialist signals were most informative.
