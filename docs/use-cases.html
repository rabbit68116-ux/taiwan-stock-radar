how to use taiwan-stock-radar in practice

This page explains the common operating modes for taiwan-stock-radar so that AI agents and human users can choose the right workflow for the task in front of them.

market sweep mode

Use this when the question is broad, such as "what should I watch in Taiwan stocks today?" The system scans the wider universe, checks regime and sector context, and surfaces the top 20 candidates for attention.

watchlist refinement mode

Use this when a shortlist already exists and the goal is to compare names more carefully. The system focuses on ranking clarity, factor breakdown, and which names deserve to move up or down in priority.

single-stock memo mode

Use this when the question is narrow, such as "analyze 2330" or "should I buy this stock now?" The workflow expands to include peer context, catalysts, risk framing, and action zones.

post-trade review mode

Use this when reviewing whether a prior signal was well-framed. The system can compare the original setup, the market environment, and whether entry or exit planning was coherent.
