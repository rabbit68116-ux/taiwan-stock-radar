常見問題Frequently Asked Questions

關於 Taiwan Stock Radar v1.7 的常見問題 Common questions about Taiwan Stock Radar v1.7

這一頁聚焦說明 `v1.7` 的 shared core、08:30 日盤報告、單股委員會、Windows desktop app，以及它不應被誤解成保證報酬的地方。

This page explains the `v1.7` shared core, the 08:30 daily brief, the single-stock committee, the Windows desktop app, and what the product should never be mistaken for.

什麼是 Taiwan Stock Radar？What is Taiwan Stock Radar?

它是一個專為台股打造的雙軌研究產品。第一條路線是 skill / CLI / docs，第二條路線是 Windows desktop app；兩者共用同一套 shared core。It is a dual-track Taiwan-stock research product. One route is the skill / CLI / docs, the other is the Windows desktop app, and both share the same core.

08:30 日盤報告會怎麼運作？How does the 08:30 brief work?

系統會整合台股夜盤、Yahoo奇摩股市、Anue 鉅亨網與美股關鍵指標，輸出 10 大重點訊息、開盤偏向、綜合評估、族群觀察與風險提示。The system combines Taiwan night-session, Yahoo Taiwan market pages, Anue headlines, and key US indicators into ten key messages, an opening bias, an overall assessment, sector watchlists, and risk flags.

單股委員會會輸出哪些內容？What does the committee output?

輸出包含 signal-engine summary、specialist briefs、style profile、strategy selection、validation scorecard、scenario tree、buy zone、entry triggers、stop、take-profit、invalidation 與 dissent。The committee outputs the signal-engine summary, specialist briefs, style profile, strategy selection, validation scorecard, scenario tree, buy zone, entry triggers, stop, take-profit, invalidation, and dissent.

Windows 版和 skill 有什麼關係？How does the Windows app relate to the skill?

Windows 版不是另一套分析系統，而是 shared core 的桌面交付層。GUI 不應複製商業邏輯，只負責呼叫 workflow 與渲染結果。The Windows app is not a separate analysis system. It is the desktop delivery layer of the shared core. The GUI should not duplicate business logic; it should only call workflows and render results.

如果沒有同日 daily brief 怎麼辦？What if the same-day brief is missing?

單股委員會 workflow 會先檢查同日 `daily_market_brief`，若不存在就先補產生，再進入個股深度分析。The single-stock committee checks for the same-day `daily_market_brief` first. If it does not exist, it generates it before entering the stock analysis.

會保證報酬嗎？Does it guarantee returns?

不會。Taiwan Stock Radar 僅供研究、教育與產品展示使用，不是投資建議，也不是任何形式的報酬保證。No. Taiwan Stock Radar is for research, education, and product demonstration only. It is not investment advice and not a guarantee of returns.
