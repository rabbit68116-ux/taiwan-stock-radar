Taiwan Stock Radar 背後的參考文件 The reference library behind Taiwan Stock Radar

這一頁整理 `v1.5` 台股研究 skill 的核心文件。它不是文件清單而已，而是公開展示頁背後的方法書架，讓訪客能看到「開市前環境評估 + 單股委員會」這套流程是如何被定義、配置與限制。

This page gathers the core documents behind the `v1.5` Taiwan-equity research skill. It shows how the combined premarket layer and single-stock committee are defined, configured, and constrained behind the public exhibit.

建議先看的文件 Recommended starting points

如果你想先理解這個 skill 如何思考、台股指標如何被組織、風格如何切換，以及最終決策文件如何成形，以下幾份文件最值得先打開。

If you want to understand how the skill thinks, how Taiwan indicators are organized, how styles are switched, and how the final packet is structured, these are the best places to start.

SKILL.md

主行為定義文件，說明單股指標委員會 workflow、風格切換與輸出紀律。

The main behavior document defining the single-stock indicator committee workflow, style switching, and output discipline.

Agent analyst blueprint

說明專家角色、會議流程、策略提名、權重整合邏輯，以及最終決策文件必須包含的段落。

Explains specialist roles, meeting protocol, strategy nomination, weighted synthesis, and the required sections of the decision packet.

Indicator catalog

定義六層台股訊號引擎、最小可用欄位與各欄位的實戰用途。

Defines the six Taiwan-stock signal engines, the minimum viable field set, and the practical use of each field.

Style weights

定義短線、波段、中長線三種交易風格的權重切換。

Defines the weighting switch for short-term, swing, and position trading styles.

Premarket brief framework

說明台股夜盤、美股趨勢、半導體領先訊號與 VIX 如何轉成隔日開盤偏向、風險提示與族群觀察。

Explains how Taiwan night-session, US-market moves, semiconductor leadership, and VIX become an opening-bias brief, risk flags, and sector watchlists.

Agent personas

定義每位 analyst 的人格、專長、權重與指標偏好。

Stores role identity, specialty, weighting, and indicator focus for each analyst persona.

Case study

展示公開版本的完整單股指標委員會輸出，從個案設定一路到失效條件。

Shows the public exhibit version of a full single-stock indicator committee memo, from case setup to invalidation.

這些文件控制什麼 What these documents govern

Overnight environment

定義夜盤、美股與風險情緒如何進入台股早盤判讀，避免單股研究脫離隔夜市況。

Defines how night-session, US-market moves, and risk tone enter the Taiwan open so the single-stock view does not detach from the overnight environment.

Specialist reasoning

定義趨勢、動能、量價、籌碼、基本面、事件、策略與驗證如何在整合前先被分開思考。

How trend, momentum, price-volume, flow, fundamentals, events, strategy, and validation reasoning are separated before synthesis.

Decision structure

定義委員會如何把分歧轉成帶有風格選擇、策略選擇、信心與失效條件的最終 verdict。

How the committee turns disagreement into a weighted verdict with style choice, strategy selection, explicit confidence, and invalidation.

Execution-aware output

定義分析如何落到買入區、停損、停利框架、驗證面板、風格權重與風險備註。

How the committee translates analysis into buy zones, stop-loss logic, take-profit framing, validation panels, style weights, and risk notes.
