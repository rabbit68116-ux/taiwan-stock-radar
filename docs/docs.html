Taiwan Stock Radar 背後的參考文件 The reference library behind Taiwan Stock Radar

這一頁整理 `v1.2` 單股多 Agent 研究 skill 的核心文件。它是公開展示頁背後的文件書架，讓訪客能看見方法如何被定義，而不只是看結果。

This page gathers the core documents behind the `v1.2` single-stock council. It is the reference shelf behind the public exhibit, showing how the method is defined instead of only showing the output.
