The reference library behind Taiwan Stock Radar

This page gathers the source documents that define how the `v1.2` single-stock council behaves. It serves as the technical reference shelf behind the public-facing exhibition pages.

Recommended starting points

These are the best documents to open first if you want to understand how the skill thinks, how the council is organized, and how the final decision packet is structured.

SKILL.md

The main behavior document. It defines the single-stock council workflow and the expected discipline of the final output.

Agent analyst blueprint

Explains the specialist roles, meeting protocol, weighted synthesis, and required sections of the final decision packet.

Agent personas

Stores role identity, personality, weight, and indicator focus for each specialist analyst inside the council.
